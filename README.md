# warcraft
Главная задача — создать интерактивный справочник, который будет работать быстро и выглядеть в стиле игры.
Что я планирую сделать:
​
База данных: Полный список из всех 24 героев и ключевых предметов игры.
​Интерактивность: Возможность переключаться между категориями без перезагрузки всей страницы.
​Система управления: Добавление функционала для «прокачки» героев и сбора инвентаря.
​Атмосферный дизайн: Оформление сайта в духе Warcraft III (темные тона, золотые акценты, «каменная» эстетика).
​
Технологический стек
​Я разрабатываю этот сайт, используя стандартные веб-технологии:
​HTML5: Структура контента.
​CSS3: Визуальное оформление и стилизация под игровую тематику.
​JavaScript (ES6+): Логика работы сайта, обработка данных и динамическое отображение элементов.

​ Как я это делаю (Пошаговый план)
​Разработка ядра: Создание структуры Single Page Application (SPA), где сайт динамически подгружает данные при клике на кнопки.
​Работа с данными: Создание отдельных файлов data.json и heroes.json для того, чтобы легко дополнять базу знаний, не переписывая основной код.
​Дизайн: Внедрение CSS-стилей, которые имитируют интерфейс игры (использование специфических цветов и анимаций наведения).
​Обработка ошибок и логика: Написание скриптов, которые связывают данные с визуальным представлением карточек, и настройка системы обработки событий (покупки/прокачки).
​🛠 Как запустить проект локально
​Скачайте все файлы проекта в одну папку.
​Используйте расширение Live Server в редакторе VS Code для корректного отображения данных через fetch (или запустите простейший локальный сервер).
​Откройте index.html в вашем браузере.
​
